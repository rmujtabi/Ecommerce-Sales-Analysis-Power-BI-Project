# Ecommerce-Sales-Analysis-Power-BI-Project
This project analyzes ecommerce sales performance using a structured dataset containing order details, customer information, payment modes, product categories, and profit metrics. The goal is to uncover actionable insights into revenue trends, customer behavior, and product performance.
I developed an interactive Power BI dashboard that enables dynamic filtering and visualization to support data-driven business decisions.

**Dataset and Dashboard used in the analysis:**
Orders.csv
Sales_Data.csv
Power BI File (Sales Data.pbix)
Dashboard Screenshot: Ecommerce Sales Analysis.png

**Objectives**

Analyze sales, quantity, profit, and AOV (Average Order Value) trends.
Identify top-performing states, categories, and customer segments.
Evaluate monthly profit fluctuations and payment mode patterns.
Build an interactive dashboard for business users to explore insights.

**Tools & Technologies**

Power BI (Data Modeling, DAX, Visualization)
Power Query
CSV Dataset
Data Cleaning and ETL

**Data Preparation & Processing**

Cleaned missing and inconsistent values using Power Query.
Created a Date Table to enable time-intelligence calculations.
Built relationships between fact and dimension tables.
Implemented DAX measures such as:
Total Sales Amount
Total Quantity Sold
Total Profit
Average Order Value (AOV)
